# 官方资源目录来源

本目录里的官方资源，来自公开站点与开源仓库的整理、筛选和中文重写。
目标不是做原站镜像，而是把原本分散的 Prompt、Skill、MCP 资源，收敛成适合在移动端浏览、搜索、收藏和复制的统一目录。

说明：
- 资源标题、摘要、步骤、模板说明和安全提示，都按本产品的中文语境做了整理，不等于原项目的逐字翻译。
- Prompt 资源优先强调“拿来就用”和“变量填空”。
- Skill 资源优先强调“解决什么问题”和“怎么接进工作流”。
- MCP 资源优先强调“适合连接什么工具”和“怎么安全接入”。
